# GitHub Desktop 로컬 폴더 설정 방법

GitHub Desktop에서 기존 프로젝트(로컬 폴더)를 연결하거나 설정하는 간단한 방법입니다.

1. **GitHub Desktop 실행**: 프로그램을 엽니다.
2. **리포지토리 추가 메뉴로 이동**: 
   - 좌측 상단의 메뉴에서 **`File`** > **`Add local repository...`** 를 클릭합니다.
   - *(단축키: `Ctrl` + `O`)*
3. **폴더 선택**:
   - `Choose...` 버튼을 클릭합니다.
   - 화면에 뜨는 탐색기 창에서 연결하고 싶은 **로컬 폴더를 선택**하고 `폴더 선택`을 누릅니다.
4. **리포지토리 추가 완료**:
   - **`Add repository`** 버튼을 클릭하여 완료합니다.

> **💡 참고:** 
> 선택한 폴더가 아직 Git 저장소로 초기화되지 않은 폴더일 경우, *"This directory does not appear to be a Git repository."* 라는 경고 문구가 나타날 수 있습니다. 
> 이 경우 바로 아래에 있는 **`create a repository`** 라는 파란색 글씨를 클릭하여 해당 폴더를 새로운 깃 리포지토리로 만들 수 있습니다.
